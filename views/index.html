Tecnologias

  • Sequelize Cli
  • Node
  • Express.js

End Points

  • GET
    • /users -- Obtem os usuários cadastrados
  • PUT
    • /user/:id -- Atualiza um usuário
  • DELETE
    • /user/:id -- Deleta um usuário
  • POST
    • /user -- Cria um usuário no banco de dados /user/authenticated --- Autentica um usuário
